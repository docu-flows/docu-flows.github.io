👥 User Roles & Permissions

Access to features is gated by role using ProtectedRoute.

Super Admin

Full system access including service-package configuration, customers and quotations.

Administrator

Organisation-wide management: roles, users, customers, quotations and reports.

Admin

Manages business info, departments, units, document types and standard users.

User

Uploads, signs, tracks and downloads their own documents; views personal dashboard.

🔄 Core Document Workflow

The primary lifecycle of a document inside DocuFlow.

→
👥 Assign Signers
→
⏳ Pending / Waiting
→
✍️ Sign (Internal / External)
→
✅ Completed
→
📥 Download / Archive

🏷️ Document Status Reference

Statuses used throughout the dashboard, lists and reports.

Draft Uploaded but not yet routed
Pending Awaiting signer action
In Progress Partially signed
Completed All signers signed
Cancelled Workflow cancelled
Expired Past expiration date
